Generador de HTML.
Este es un pequeño proyecto, realizado como una automatización para un usuario.
El problema radicaba en que los usuario realizaban un proceso un tanto rebuscado.
Con ChatGPT y generar codigo HTML lo cual no comprendian.
Por lo cual se realizó una abstracción de su plantilla y se realizó una automatización en generar componentes en base a su plantilla.
